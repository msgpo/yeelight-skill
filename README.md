## yeelight-skill
Control Xiaomi Yeelight with Mycroft

## Description 
Utilize the Yeelight API and Python library for controlling YeeLight WiFi RGB bulbs with Mycroft.
## Examples 
* "turn lights on"
* "turn lights off"
* "dim the lights"
